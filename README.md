This is just a simple little Discord bot that waits for someone to use the word "farts" and replies by saying "Haha you said farts"

To run it just execute `python run_farts_bot.py`. But before you do, you'll need to make an application in the [Discord developer portal](https://discord.com/developers/applications) and make a new .env file that includes its app key like this:

```
DISCORD_APP_KEY="whatever_your_key_is"
```
